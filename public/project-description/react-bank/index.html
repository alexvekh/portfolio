Description

The Bank application was developed according to the design provided by course IT_BRAINS as the final project.

This is a full-stack web application employing cutting-edge technologies such as React for the front-end and Node.js/Express for the back-end.

This comprehensive application empowered users to perform a spectrum of tasks, including registration with code confirmation, login, logout, password changes, and the restoration of forgotten passwords.

Authenticated users gained access to personal account balances, transaction history, and notifications, enabling them to simulate transactions between pay systems and transfer money to other accounts.

Rigorous input form validations were applied throughout the application to ensure data integrity.

Key Contributions

  • Leveraged React to craft an engaging and interactive user interface.
  • Engineered the back-end system using Node.js and Express, ensuring robust functionality.
  • Employed a variety of front-end and back-end technologies to manage user-specific data and implement validation logic.
  • Designed an intuitive and responsive user experience with CSS/SASS.
